# 🎵 Spotify Genre Segmentation

## 📌 Overview
This project uses Machine Learning to classify and segment songs into genres using Spotify audio features.

## ⚙️ Features Used
- Tempo
- Energy
- Danceability
- Loudness
- Acousticness
- Valence

## 🧠 Models Used
- K-Means Clustering
- Random Forest Classifier

## 📊 Results
- Achieved accurate genre classification
- Identified patterns in music attributes
